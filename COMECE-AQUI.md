## ⚡ 5 MINUTOS PARA COMEÇAR

### 1. Abrir a Landing Page

**No seu computador:**
1. Vá para pasta: `c:\Users\kinam\Desktop\teste\negocio-digital-7-dias`
2. Clique com botão direito em `index.html`
3. Abra com → Google Chrome (ou outro navegador)

**Pronto!** Você já tem uma landing page funcionando.

### 2. Personalizar a Landing Page

Abra `index.html` com um editor de texto (Notepad++, VS Code):

**Mude o preço:**
```html
<span class="price-currency">5.000</span> Kz
```
Altere `5.000` para o preço que deseja.

**Mude o título:**
```html
<h1>
    Negócio
    <br>
    <span class="highlight">Digital</span>
    <br>
    em 7 Dias
</h1>
```

**Mude descrição:**
Procure por `<p class="hero-subtitle">` e altere o texto.

**Mude imagem:**
Procure por `.placeholder-image` e troque por sua imagem:
```html
<img src="sua-imagem.jpg" alt="Seu negócio digital">
```

### 3. Ler o E-book

**Arquivo:** `content/ebook.md`

Opções:
1. **Ler no navegador:** Copie conteúdo → Cole em app nota
2. **Converter em PDF:** Abra → Ctrl+P → Save as PDF
3. **Ler direto:** Abra com editor de texto qualquer

## 📊 Como Acompanhar Progresso

### Criar Planilha de Rastreamento

1. Vá para **Google Sheets** (sheets.google.com)
2. Crie nova planilha
3. Copie este modelo:

```
DATA | NOME | CONTATO | RESPOSTA | PROPOSTA | VALOR | PAGO?
-----|------|---------|----------|----------|-------|-------
01/01| João | 923456789| Sim | Sim | 20.000| Não
```

### Acompanhe Estas Métricas

- **Contatos/dia:** Quantas pessoas contatou
- **Taxa de resposta:** % que respondeu
- **Taxa de venda:** % que virou cliente
- **Valor médio:** Quanto ganha por cliente

**Fórmula:** Se contatar 20 pessoas, 1 vira cliente.
