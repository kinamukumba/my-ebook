# 50+ Prompts de IA Para Seu Negócio Digital

Prompts prontos para copiar, colar e customizar no ChatGPT, Gemini ou similares.

## PROMPTS PARA COPYWRITING

### 1. Descrição de Serviço Persuasiva

```
Você é copywriter especializado em vender serviços digitais.
Escreva uma descrição de 3 linhas para meu serviço: [DESCREVER SEU SERVIÇO]
Público-alvo: [QUEM COMPRA]
Principal benefício: [RESULTADO PRINCIPAL]
Tom: direto, confiante, sem jargão técnico, em português-angolano.
```

### 2. Título Irresistível

```
Crie 5 títulos persuasivos para meu serviço: [SEU SERVIÇO]
Cada título deve:
- Deixar claro o RESULTADO (não o processo)
- Incluir número específico ou promessa medível
- Ser irresistível para: [PÚBLICO-ALVO]
- Gerar urgência ou curiosidade
Formato: Lista numerada com 1-2 linhas de explicação cada.
```

### 3. Argumento de Venda (Pitch)

```
Crie um argumento de venda de 10 linhas para [SEU SERVIÇO].
Estrutura:
1. Problema do cliente (2 linhas)
2. Sua solução específica (2 linhas)
3. Resultado que vão alcançar (2 linhas)
4. Prova/evidência (2 linhas)
5. Call-to-action (2 linhas)
Tom: conversacional, como falando com amigo, em português-angolano.
```

### 4. Prop de Venda em Uma Linha

```
Resuma a proposta de valor de [SEU SERVIÇO] em UMA linha que seja irresistível.
Deve começar com ação: "Vou..." ou "Você vai..."
Exemplo: "Vou triplicar seus seguidores em 30 dias ou seu dinheiro volta"
Crie 5 opções para eu escolher.
```

### 5. Descrição de Landing Page

```
Você é expert em landing pages que convertem.
Escreva seção "hero" para minha landing page de [SEU SERVIÇO].
Incluir:
- Título em 2-3 palavras
- Subtítulo de 1-2 linhas
- 3-4 benefícios principais (em bullets)
- Chamada para ação clara
Público: [DESCREVER]
Tom: urgente, inspirador, possível.
```

## COMO USAR ESSES PROMPTS

1. **Copie o prompt inteiro** (entre as ```)
2. **Cole no ChatGPT/Gemini**
3. **Customize** [ENTRE COLCHETES]
4. **Pressione Enter**
5. **Refine** a resposta se necessário

## DICAS PARA MELHORES RESULTADOS

- **Seja específico** - Quanto mais detalhes, melhor resultado
- **Experimente** - Teste diferentes versões
- **Refine** - Peça "Faça isso mais [curto/longo/persuasivo]"
- **Combine** - Pegue parte de um prompt + parte de outro
- **Personalise** - Adapte para sua voz/marca
